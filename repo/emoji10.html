Which does what?

EmojiAttributes

This tweak helps display emoji characters properly. Otherwise, we may see "?", "U", square or just blank space instead of actual emojis.

EmojiLocalization

This tweak adds missing localized strings on emoji keyboard.

EmojiResources

This tweak adds missing category icons on emoji keyboard.

EmojiFontManager

This tweak allows users to install correct emoji font without filesystem modification.

Troubleshooting

General concern

As far as I know, many people don't change the emoji font on their devices to the proper one after they install the tweaks. Since the original fonts won't contain any new emoji, users experience bugs at displaying emoji characters instead.

Issues with font

BytaFont (well, any version) is known to cause problems more than profits because it has to swap a new emoji font with the default one. If the swap fails or there is no enough disk space for the font, emoji keyboard is messed up.

To get around this problem, you have to see what happened inside the directory /System/Library/Fonts/Core (iOS 8.3+) or /System/Library/Fonts/Cache otherwise. The original font should still be there, but maybe with a wrong file extension.

  • For iOS 6.0 - 8.2, the exactly correct name of the emoji font is AppleColorEmoji@2x.ttf (retina devices) or AppleColorEmoji.ttf otherwise
  • For iOS 8.3 - 9.3, similar to the above but you will also have a file with the same name but with extension .ccf

If you see something different, you are likely to have issues with the font. To restore default fonts, there are multiple ways:

  • Get BytaFont Backup fonts that should be in Cydia for you recover lost fonts
  • /User/Library/Preferences/BytaFontBackup should save all default fonts (provided that you use BytaFont), you can get the old emoji font from there
  • I do provide some system-default emoji fonts here that you may use
  • There are also some links to the font that you may find on the internet, or reddit /r/jailbreak

Technically, by installing the correct font, this should fix 90% of problems installing Emoji10 that new users usually encounter.

Emoji duplication bug

This problem can occur on iOS 8.3 and above with Emoji10 installation. The latest versions already prevent that from happening, but users may be able to see just because they used old versions. The thing is this bug persists even after uninstallation. To fix it, you can either:

  • Delete /User/Library/Preferences/com.apple.EmojiPreferences.plist and respring your device
  • Tap Reset emoji preferences in the settings page of EmojiFontManager, if you use it
